# WSL 기반 빌드 방법 (PyInstaller)

이 문서는 Windows에서 WSL을 사용해 리눅스용 단일 실행 파일을 만드는 방법을 정리한 것이다.

## 1) WSL 접속
Windows 터미널 또는 PowerShell에서 WSL을 실행한다.
```bash
wsl
```

## 2) 작업 경로 이동
Windows의 D: 드라이브는 WSL에서 `/mnt/d`로 접근한다.
```bash
cd /mnt/d/Work/testScript/linux-checks
```

## 3) Python/pip 확인 및 설치
```bash
python3 --version
```

`pip`가 없으면 설치한다.
```bash
sudo apt update
sudo apt install -y python3-pip
```

## 4) PyInstaller 설치
```bash
python3 -m pip install --user pyinstaller
```

## 5) 빌드 실행
스크립트 사용:
```bash
./build_pyinstaller.sh
```

직접 실행도 가능:
```bash
python3 -m PyInstaller --onefile check_runner.py
```

## 6) 결과물 확인
```bash
ls -l dist/check_runner
```

생성된 `dist/check_runner`가 리눅스 실행 파일이다. 서버로 옮겨 실행하면 된다.

## 7) 서버에서 실행 방법
서버로 `dist/check_runner`를 옮긴 후 아래처럼 실행한다.
```bash
chmod +x check_runner
sudo ./check_runner
```

`sudo` 없이 실행하면 일부 점검(예: `jps -l`, `systemctl`)이 실패할 수 있다.
